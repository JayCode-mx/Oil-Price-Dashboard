Built a Real-Time API Monitoring Dashboard 🚀

I wanted to understand how systems detect outages and latency spikes in real-world applications.

So I built:

⚡ A dashboard that tracks API latency
🧠 Detects anomalies using EWMA logic
📊 Generates insights on best/worst usage times
🚨 Identifies network-wide issues vs service issues

Tech stack:
React + Node.js + Tailwind + Render
